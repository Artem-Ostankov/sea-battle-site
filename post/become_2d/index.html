Become 2D

A 2 minute read, Posted on Fri, Apr 28, 2017

Hi ppl !

New Friday came, and new update follows.
So there are 2 big changes.
Again all based on your mega-awesome feedback ! Thanks a lot for that ! :)

Change #1: Networking
Feedback: This would be awesome to play this game easy on website.
There are several problems here:
- I picked standard unity networking engine: UNet.
- in UNet connection matchmaking is somewhat slow.
- UNet feels quite limited in terms of extra features and integration outside of Unity
- It’s not that good on WebGL, because it falls back to WebSocets and they are not that much supported in UNet.

So we go to Photon! Actually I wanted to use it from the beginning but felt that UNet grown better… well it appears that Unet is still lugging behind Photon.
And Photon is awesome !
It has a lot of nice features, like web-hooks and server side Room Data keeping.
It means, I can change room setting outside of Unity and can keep room alive even if both player disconnected, so that they can reconnect anytime.
This is very cool and quite easy to setup, so I’m really happy !

Change #2: 3D -> 2D
Current tests showed that game is relatively slow to load in a browser.
Given that the game is not really Half-Life 2 or Crysis … and game looks really lightweight, so should actually be lightweight.
Therefore, I tried to figure out if there really is any feature that requires me to keep it 3D.
The answer is NO !
So wtf, let’s make it 2D :-)
This brings quite good advantages:
- size (faster load)
- faster rendering, esp. on mobile and browser
- potentially even better, nicer appearance than in 3D

So far I did a bit of the progress on both of the sides:
Networking: get on trails with Photon - worked through the several tutorials, docs and demos.
2D: made 1st changes towards 2D : board(sea), simple turret, crosshair and selector.

Will continue working on these changes over the week, hopefully should be done by next Friday !
