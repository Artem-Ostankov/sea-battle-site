Raw game mechanics is not enough

UI matters even in a early phase :-/

A 2 minute read, Posted on Thu, Mar 2, 2017

Current state: - WEGO mechanics - Hot-Seat only multiplayer - 4 types of units: Cannon, Shield, GatherMoney, Neutral - Basic UI with buttons for main functionality - Fog Of War visibility mechanics (still might need some refactoring)

So all in all, game is in it’s quite basic form but seems to work!

Current goal: bring it to people to start play testing.

The reason no one is play testing right now is that I have already first couple of feedbacks.

Main outcomes of this feedback were: - People do need clear and intuitive UI to concentrate on a game itself - There should be some sort of “Help” so that I spend less time explaining things ( maybe I should include also small video ? ) - there should be visual feeling of things happening. - the best if that would be playable over internet (what? on this stage?!)

This basically means that unfortunately I can’t really give it for playtest right now. As you see from top, nothing tells about gameplay itself, it’s more about general usability of the app. I need playtesters to concentrate they experience on the game itself and therefore I need to eliminate all the surrounding hassle, so they can just play.

So here it goes, the next main goals before I can publish my demo/alpha/beta/whatever game.

For sure: - Main menu with buttons: for sure: - Hot-sit - Exit maybe: - for help video (maybe) - setting - internet game - Make sure it’s not buggy and mechanics works as suppose (all units works fine) - Adjust land tiles (colors etc.) - Help panel (show what unit does what, and that color of the unit depends on the land it stands on) - Website for game: simplest: just put in as I already did: in my site under www.ostankov.com/game better: come-up with name and create a proper separate website with to start generating traffic

Maybe: Actions player Go back / forward in turns Come-up with name Separate website Online multiplayer

OK, goals are clear, so back to coding !
