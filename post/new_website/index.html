New Web Site

I riched the point when sharing game binaries through skype or drobbox became just timeconsuming. So I thought, it's a good moment to finally lunch a simple website where I would keep and share all versions of the game. My last own "site" (www.ostankov.com) was built with HUGO (www.hugoio.com) and I got quite a good experience using that. Though I still felt a bit uncomrtable wtih HUGO templating DSL. So I've decided to look for something more familiar and expressive in area of static site generators. >>> Small step-a-side, static site generators - are frameworks where you pregenerate all pages and deploy it as simple statis website. There are huge awantages: you can publish it on any simple web hosting, you can use CDN very effectevly as it's gonna cache all your pages, it's very fast and therefore SEO optimized, all the changes to your blog will be versioned and kept as code (opposed to MySql for wordpress). As in general I'm big fun of Python, this was my first pick to look for an alternative. The most known static site generator in Python is Pelican. It has a decent development history and lots of plugins. I spent a 1-2 days hacking through it to throw it away by the end. The main problem a lot of Python projects have is the quality of the code and legacy. Bad quality comes from little barier language puts, so basically any 1st-grad can start hacking on it. Legacy is different issue, as Python has already gone through it's 'cool-kid' age and most of well developed projects lost it's momentum and stuck in some old decissions. Combined .... well Pelican being well more mature then Hugo lacks many features and easiness of newer framework. So all togather I decided to stick with Hugo. By now it's fairly simple website with Home(/), Posts (/post/), Downloads(/download/). It's conent is fully generated with GoLang templates from MarkDown files. Another point to think about was where to store all the binaries, as one binary for one platform already takes around 20-30 MB. Here another great advantage of Unity came in play: unity cloud build. It's an awesome and *free* service from Unity to Build and Share executables. So all in all, please wellcome a new kid in a city: Sea Battles Evolved ! PS. Name is a working title and is still up for descussion :P
