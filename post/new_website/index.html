New Web Site

New Webisite for an indie game - Sea Battle Evolved.

A 2 minute read, Posted on Mon, Apr 24, 2017

I riched the point when sharing game binaries through skype or dropbox became just time-consuming. So I thought, it’s a good moment to finally launch a simple website where I would keep and share all versions of the game.

My last own site was built with HUGO and I got quite a good experience using that. Though I still felt a bit uncomfortable with HUGO templating DSL. So I’ve decided to look for something more familiar and expressive.

Small step-a-side, static site generators â€” are frameworks where you pre-generate all pages and deploy it as a simple static website. There are huge advantages: you can publish it on any simple web hosting, you can use CDN very effectively as it’s gonna cache all your pages, it’s very fast and therefore SEO optimized, all the changes to your blog will be versioned and kept as a code (opposed to MySql for WordPress).

As in general, I’m a big fan of Python, this was my first pick to look for as an alternative. The most known static site generator in Python is Pelican. It has a decent development history and lots of plugins. I spent 1-2 days hacking through it to throw it away at the end. The main problem a lot of Python projects have is the quality of the code and legacy. Bad quality comes from little barrier language has, so basically any 1st-grad can start hacking on it. Legacy is a different issue, as Python has already gone through it’s ‘cool-kid’ age and most of the well-developed projects lost it’s momentum and stuck in some old decisions. Combined …. well Pelican being more mature then Hugo lacks many features and easiness of a newer framework. So all together I decided to stick with Hugo.

Within a several hours I got fairly simple website with Home, Posts and Downloads. Its content is fully generated with GoLang templates from MarkDown files.

Another point to think about was where to store all the binaries, as one binary for one platform already takes around 20-30 MB. Here another great advantage of Unity came in play: unity cloud build. It’s an awesome and free service from Unity to Build and Share executables.

So all in all, please welcome a new kid in a city: Sea Battles Evolved !
