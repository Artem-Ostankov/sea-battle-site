Simultaneous vs Ordered

too much action

A 1 minute read, Posted on Tue, Apr 25, 2017

Hi ppl !

I worked through last comments of yours for the current version. The most common feedback was about simultaneous execution of all the actions. You guys said that during the execution phase it was quite hard to follow what is going on.

So I’ve built a new version where this mechanics is changed a bit. In this new version actions are played in the order you committed them. So as first yours and opponents 1st action will play, then yours and opponents 2nd action … and so forth. Still, the natural sub-sequence of the actions is taken into account in each of the pair. By that I mean the following order: Stun -> Shield -> Rocket hit. This approach also opens very cool possibilities. For example while planning actions you will see what actions were already committed by the opponent and therefore you can plan a better counter-action. Also, I can introduce a new type of action, that can be applied not to the field but to the card. For example “Spy” action -> it will disclose the target for a selected opponent’s action, so that you can effectively block it.
